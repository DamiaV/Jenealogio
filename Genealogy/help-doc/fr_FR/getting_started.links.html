Links

A link is the graphical representation of the marriage between two people. It is a line that connects two cards. The default color is black. If the couple has/had children, a line for each child is drawn (default color: blue).
Example:

Here, the black line is the link representing the marriage and the blue line connects to the (only) child the couple has.

When a link is hovered by the mouse, it becomes red by default. A selected link will appear green (default color).
Colors can be changed in the Options menu.

The next three topics will explain how to add, edit or delete links.
