Adding links

To add a link, follow one of these methods:

  • go to Edit > Add Link...;
  • or click Add Link...;
  • or press Ctrl+L
Now, select the first card (if it is not already) you want to connect, then click the second one. You should now be seeing this form:

The first two fields (1) are the date and location of the wedding.
The two fields below (2) contain the names of the couple.
Zone (3) lets you add/remove children from the marriage. To add children, select the items from the bottom list then click the button. To remove added children, select the items in the top list then click the button.

Once you're finished, click Validate to create the link. If you clicked before finishing, don't worry you can come back and edit it later.

Now, there should be line between the two cards and the children.
